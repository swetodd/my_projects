# RSA Криптосистема на C++

## Описание проекта
Реализация асимметричного шифрования RSA с использованием классов для работы с большими числами и криптографическими операциями.

## Архитектура
Проект состоит из двух основных классов:

### 1. **BigInteger** - класс для работы с большими числами
- Хранение чисел произвольной точности
- Арифметические операции (+, -, *, /, %)
- Операции сравнения
- Модульное возведение в степень
- Расширенный алгоритм Евклида
- Тест Миллера-Рабина на простоту
- Генерация случайных простых чисел

### 2. **RSA** - класс для реализации алгоритма RSA
- Генерация ключевой пары
- Шифрование/дешифрование чисел и текста
- Сохранение/загрузка ключей
- Валидация ключей

cd build

make
./rsa_crypto
